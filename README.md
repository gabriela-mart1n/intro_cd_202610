## ⚙️ Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/tuusuario/mi-proyecto.git
```

2. Entra a la carpeta:
```bash
cd mi-proyecto
```

3. Instala las dependencias:
```bash
npm install
```

4. Configura las variables de entorno:
```bash
cp .env.example .env
```

5. Inicia el proyecto:
```bash
npm run dev
```
